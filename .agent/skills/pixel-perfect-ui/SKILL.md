# Pixel-Perfect UI Enforcement Skill  

**Goal:** مقارنة الواجهة الحالية مع التصاميم وضمان توافق التفاصيل (مسافات، ألوان، خطوط) حتى المستوى البكسلي.  

**Instructions:**  
- قارن لقطات (Screenshots) أو التصميمات الرسمية مع الواجهة الحالية.  
- تحقق من تطابق الهامش (margin)، التعبئة (padding)، وحجم الخط واللون مع مواصفات التصميم (مثلاً فئات Tailwind ذات القيم الثابتة).  
- اذكر أي فوارق (مثل لون خاطئ، تباعد غير متناسق) واقترح تعديل CSS أو الفئات لتصحيحها.
