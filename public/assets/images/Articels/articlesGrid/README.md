# Articles Grid Images

This folder should contain the article images for the Articles Grid section.

## Required Images

Please add the following images to this folder:

1. **article1.webp** - Industrial Sector (Environmental Impact Studies)
   - Recommended size: 780x580 pixels
   - Image showing industrial facilities/factories

2. **article2.webp** - Real Estate and Tourism Sector (Environmental Management Plans)
   - Recommended size: 780x580 pixels
   - Image showing modern buildings with green elements

3. **article3.webp** - Infrastructure (Audit and Rehabilitation)
   - Recommended size: 780x580 pixels
   - Image showing quarry or infrastructure site

4. **article4.webp** - Renewable Energy (Compliance and Monitoring)
   - Recommended size: 780x580 pixels
   - Image showing renewable energy installations (wind turbines, solar panels)

## Image Specifications

- **Format**: WebP (preferred for web optimization)
- **Dimensions**: 780 x 580 pixels (4:3 aspect ratio)
- **Quality**: High quality, optimized for web
- **File Naming**: Use lowercase with no spaces (article1.webp, article2.webp, etc.)

## Temporary Placeholder

Until you add the actual images, you can use placeholder images from:
- Unsplash (https://unsplash.com)

## How the Images are Used

These images are displayed in the ArticlesGrid component with alternating layouts:
- Article 1 & 3: Image on the left, content on the right
- Article 2 & 4: Content on the left, image on the right

The images have a rounded corner design (24px border radius) and are fully responsive.
